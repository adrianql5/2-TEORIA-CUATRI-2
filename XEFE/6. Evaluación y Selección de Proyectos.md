# 6.1 Concepto de inversi√≥n

# 6.4 Valor Actual Neto (VAN)
Es el valor actualizado de los beneficios netos futuros de una inversi√≥n **menos el desembolso inicial**. Pasos para calcular el VAN
1. Actualizar los flujos 

$$\textbf{VAN} = -A + \frac{Q_1}{(1+k)^1} + \frac{Q_2}{(1+k)^2} + \dots + \frac{Q_n}{(1+k)^n}$$

Donde:
- **A** = inversi√≥n inicial
- **Q‚ÇÅ, Q‚ÇÇ, ..., Q‚Çô** = flujos netos de caja futuros
- **k** = tasa de descuento

### **Pasos para calcular el VAN**
Pasos para calcular el VAN
1. Actualizar los flujos 1. **Actualizar** los flujos netos de caja al momento 0 usando la tasa de descuento (**k**).
2. **Sumar** los valores actualizados ‚Üí se obtiene el **Valor Actual (VA)**.
3. **Restar** la inversi√≥n inicial ‚Üí se obtiene el **VAN**.

### **Interpretaci√≥n del VAN**
- **VAN > 0**: la inversi√≥n **genera valor** ‚Üí **Aceptar**
- **VAN < 0**: la inversi√≥n **destruye valor** ‚Üí **Rechazar**
- **VAN = 0**: ni gana ni pierde ‚Üí **Indiferente**

### **Influencia de la tasa de descuento (k)**

> A mayor tasa de descuento, **menor VAN**: el valor del dinero futuro disminuye.


### **Caracter√≠sticas del VAN**
- Es un **criterio claro y directo** para evaluar inversiones.
- Mide la **rentabilidad absoluta** (en ‚Ç¨).
- Es **aditivo**: el VAN de varios proyectos es la suma de sus VAN individuales.

# 6.5 Tasa Interna de Rentabilidad (TIR)

$$-A + \frac{Q_1}{1 + r} + \frac{Q_2}{(1 + r)^2} + \dots + \frac{Q_n}{(1 + r)^n} = 0 \rightarrow r$$

### **¬øQu√© indica la TIR?**

### **Criterio de decisi√≥n usando la TIR**
